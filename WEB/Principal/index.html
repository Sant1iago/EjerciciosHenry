Para formularios se utiliza el codigo "Form" y despues un "Input"

el tipo "submit" es "enviar" y si no tiene un valor, es el que se le asigna,
un valor es el nombre que va a tener el boton, hablando de ello se puede agregar un boton
y funciona igual a "submit", el "required" es requerido. Codigo "textarea" es para
tener un lugar mas para escribir pero que se puede modificar su tamaño, si no se quiere
dejar modificar se usa el "readonly="""


metadatos:
Se usa la etiqueta "meta" sirve apra cambiar la codificacion de la página y se pone en el HEAD, no en el BODY
se agrega el "utf-8" con "charset" (conjunto de caracteres) para que agregue la bibilioteca se caracteres especiales

Además de esas etiquetas se pueden usar las de "name = "keywords" content="palabra clave 1, palabra clave 2, ..."" que es para que ayude en la busqueda de la persona
y la web muestre nuestra pagina
otra es "name = "description" content = "descripcion de la pagina"" la cual nos sirve para hacer una descripcion del contenido de la pagina de mas o menos 70 a 140 caracteres

con "name = "author" content = "nombre del autor""
"name = "copyright" content = "nombre de la empresa dueña del dominio""
"name = "robots" content = "index" es para que muestre nuestra pagina en el indice de busqueda cuando alguien busque tanto las palabras claves br como la descripcion y "noindex" es su opuesto"
